 Memory Card Game
A modern Memory Card Game built using HTML, CSS, and JavaScript.

🚀 Features
Multiple difficulty levels
Timer system
Score tracking
Leaderboard using Local Storage
Sound effects and background music
Responsive design
Pause and restart functionality
Achievement system
Modern gaming UI
🛠 Technologies Used
HTML5
CSS3
JavaScript
🎯 What I Learned
DOM Manipulation
Game Logic
State Management
Event Listeners
Local Storage
Responsive Design
JavaScript Timers
Debugging and Problem Solving
